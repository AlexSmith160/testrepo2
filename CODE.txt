Функция ДополнитьМассив(ИсходныйМассив, КоллекцияДобавляемыхЭлементов = Неопределено) Экспорт

    Если КоллекцияДобавляемыхЭлементов = Неопределено Тогда
        Возврат ИсходныйМассив;
    КонецЕсли;

    Для Каждого _Элемент Из КоллекцияДобавляемыхЭлементов Цикл
        ИсходныйМассив.Добавить(_Элемент);
    КонецЦикла;

    Возврат ИсходныйМассив;

КонецФункции
